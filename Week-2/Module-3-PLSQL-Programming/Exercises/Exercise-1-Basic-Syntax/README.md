# Exercise 1 - Basic Syntax

Objective:
Understand the structure of a PL/SQL block and variable declaration.

Approach:
Created an anonymous block, declared variables, assigned values, and displayed the output using DBMS_OUTPUT.

Sample Output:
Student Name : B Teja Reddy
Student ID : 2027001

Learning Outcome:
Learned how variables are declared and how a PL/SQL block executes.

Note:
This exercise was executed using Oracle SQL Developer and Oracle Database XE.

How to Execute:
1. Open Oracle SQL Developer.
2. Create a new SQL Worksheet.
3. Copy the script.
4. Run using Ctrl + Enter or F5.
5. Verify the output.
