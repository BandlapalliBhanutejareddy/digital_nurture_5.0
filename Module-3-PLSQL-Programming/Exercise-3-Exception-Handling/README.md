# Exercise 3 - Exception Handling

Objective:
Learn how to handle runtime errors using exception blocks.

Approach:
Created a PL/SQL block that attempts division by zero and catches the ZERO_DIVIDE exception.

Sample Output:
Cannot divide by zero.

Learning Outcome:
Understood exception handling and graceful error reporting in PL/SQL.

Note:
This exercise was executed using Oracle SQL Developer and Oracle Database XE.

How to Execute:
1. Open Oracle SQL Developer.
2. Create a new SQL Worksheet.
3. Copy the script.
4. Run using Ctrl + Enter or F5.
5. Verify the output.
