# Exercise 2 - Control Structures

Objective:
Learn how to use IF-ELSIF-ELSE statements in PL/SQL.

Approach:
Wrote a simple grade calculator that evaluates a numeric score and prints the appropriate grade.

Sample Output:
Grade B

Learning Outcome:
Understood control flow decisions using conditional statements.

Note:
This exercise was executed using Oracle SQL Developer and Oracle Database XE.

How to Execute:
1. Open Oracle SQL Developer.
2. Create a new SQL Worksheet.
3. Copy the script.
4. Run using Ctrl + Enter or F5.
5. Verify the output.
