# Exercise 4 - Cursors

Objective:
Learn how to use explicit cursors to fetch and process rows from a table.

Approach:
Created a sample employees table, inserted a few records, and used a cursor to fetch and display each row.

Sample Output:
101 - Rahul
102 - Priya
103 - Kiran

Learning Outcome:
Understood cursor handling and row-by-row data processing in PL/SQL.

Note:
This exercise was executed using Oracle SQL Developer and Oracle Database XE.

How to Execute:
1. Open Oracle SQL Developer.
2. Create a new SQL Worksheet.
3. Copy the script.
4. Run using Ctrl + Enter or F5.
5. Verify the output.
