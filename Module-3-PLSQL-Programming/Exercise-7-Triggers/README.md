# Exercise 7 - Triggers

Objective:
Learn how to create database triggers in PL/SQL.

Approach:
Created an audit table and a trigger that inserts a record into the audit table after each employee insert.

Sample Output:
A new row is added to employee_audit after an insert into employees.

Learning Outcome:
Understood trigger execution and event-driven database programming.

Note:
This exercise was executed using Oracle SQL Developer and Oracle Database XE.

How to Execute:
1. Open Oracle SQL Developer.
2. Create a new SQL Worksheet.
3. Copy the script.
4. Run using Ctrl + Enter or F5.
5. Verify the output.
