# Exercise 6 - Packages

Objective:
Learn how to group related PL/SQL objects using packages.

Approach:
Created a package specification and body that define and implement a procedure for displaying a message.

Sample Output:
Student Package Executed

Learning Outcome:
Understood package structure and modular organization of PL/SQL code.

Note:
This exercise was executed using Oracle SQL Developer and Oracle Database XE.

How to Execute:
1. Open Oracle SQL Developer.
2. Create a new SQL Worksheet.
3. Copy the script.
4. Run using Ctrl + Enter or F5.
5. Verify the output.
