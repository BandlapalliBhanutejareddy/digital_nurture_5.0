# Exercise 5 - Procedures and Functions

Objective:
Learn how to create and use procedures and functions in PL/SQL.

Approach:
Created a stored procedure to print a message and a function that returns the square of a number.

Sample Output:
Welcome to PL/SQL Programming

Learning Outcome:
Understood modular programming using procedures and functions.

Note:
This exercise was executed using Oracle SQL Developer and Oracle Database XE.

How to Execute:
1. Open Oracle SQL Developer.
2. Create a new SQL Worksheet.
3. Copy the script.
4. Run using Ctrl + Enter or F5.
5. Verify the output.
