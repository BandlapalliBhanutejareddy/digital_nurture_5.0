# Module 4 - Test Driven Development and Logging Framework

This module contains simple Java exercises for JUnit 5, lifecycle testing, Mockito, SLF4J logging, and Lombok. I worked through these examples to understand how tests and logging help make Java programs easier to verify and maintain.

## Structure

- Exercise 1: JUnit Basics
- Exercise 2: JUnit Lifecycle
- Exercise 3: Mockito
- Exercise 4: SLF4J Logging
- Exercise 5: Lombok
- Practice/BankingCalculator/

## Learning Outcome

While working on this exercise, I understood how JUnit helps verify program correctness without manually checking every output. I also learned that logging makes it easier to follow what happens during execution.
